This project searches movies and tv shows, then displays streaming sources if found. User registration allows for creation of a personalized watchlist with sources displayed. System pieces include:

  • Linux, Nginx/Gunicorn, MongoDB, Python/Flask
  • Webserver + framework: Nginx/Gunicorn, Python/Flask
  • Database: MongoDB
  • HTML Templating: jinja, VueJS
  • UI: Bootstrap, some custom CSS
  • DevOps: AWS, Ansible, ufw
  • Other tools: flash messaging, wtforms, logging, REST, APIs
  • Dev Environment: Ubuntu, vim/tmux, github
