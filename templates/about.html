This project searches movies and tv shows, then displays streaming sources if found. User registration allows for creation of a personalized watchlist with sources displayed. System pieces include:

  • Application: display multiple results, display media detail, add/remove media from a user's watchlist
  • Webframework: flask with jinja templates
  • Database: mongodb, access via restful endpoints
  • APIs: themoviedb, amazon
  • UI: bootstrap css/js framework, flash messaging, wtforms, custom css box wrapping
  • Webserver: gunicorn
  • Dev Environment: python2.7, github, vim, tmux, virtualenv, mobaxterm ssh
  • Networking: ufw firewall port access, and iptables port redirect
  • Hosting: Digital Ocean, ubuntu
  • Logging: python built-in logging
